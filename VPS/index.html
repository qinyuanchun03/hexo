方法1：XUI面板

一键安装&升级

手动安装XUI面板

  1. 首先在 这里下载最新的压缩包，一般的VPS多为AMD64架构

  2. 在服务器解压这个包，可在下载页面，对着文件右键，复制链接。输入wget 你复制的链接即可直接在服务器下载

  3. cd /root/确定到你下载的目录

  4. 随后一步步复制代码执行

  5. rm x-ui/ /usr/local/x-ui/ /usr/bin/x-ui -rf
tar zxvf x-ui-linux-amd64.tar.gz
chmod +x x-ui/x-ui x-ui/bin/xray-linux-* x-ui/x-ui.sh
cp x-ui/x-ui.sh /usr/bin/x-ui
cp -f x-ui/x-ui.service /etc/systemd/system/
mv x-ui/ /usr/local/
systemctl daemon-reload
systemctl enable x-ui
systemctl restart x-ui

curl -fsSL https://get.docker.com | sh

mkdir x-ui && cd x-ui
docker run -itd --network=host \
 -v $PWD/db/:/etc/x-ui/ \
 -v $PWD/cert/:/root/cert/ \
 --name x-ui --restart=unless-stopped \
 enwaiax/x-ui:latest

docker build -t x-ui .


## 访问你的面板

1. 首先你买完放服务器会得到一个公网IP，这就是你访问面板的IP
2. 安装xui面板会询问你开在哪个端口，可自行输入。默认为54321
3. 在浏览器地址栏输入：ip:54321，即可访问
4. 默认密码为admin
5. 可使用CF的域名证书

## 申请域名acme证书

来到cloudflare，登录你的账号

需满足以下条件

- 知晓Cloudflare 注册邮箱
- 知晓Cloudflare Global API Key
- 域名已通过cloudflare进行解析到当前服务器

- 如没有域名，请看这个：[免费域名注册教程](https://blog1.moshangyanyu.us.kg/email-to-chatgpt/)

- 可在作者原仓库查看方法：[作者仓库](https://github.com/vaxilu/x-ui)

注意事项:

- 该脚本使用DNS API进行证书申请
- 默认使用Let'sEncrypt作为CA方
- 证书安装目录为/root/cert目录
- 本脚本申请证书均为泛域名证书

## 申请CF证书

登录你的cf，点击任意一个你注册的域名

随后开始步骤

1. 首先点击 **ssl/tls**
2. 更改概述部分的tls设置，点击配置，改为完全
3. 随后在展开的左侧菜单，点击源服务器，创建一个证书
4. 按默认设置点击下一步，复制证书内容保存即可

## CF设置部分

### 请先看如何访问面板

IP:端口

点击添加入站，选择Vless，端口选择2053，保存

即在默认情况下在你服务器的2053端口开放了一个节点，这个节点可以保存到你的手机和电脑

### 给域名添加一个解析记录

如果你通过前面的教程申请了域名，并托管到了cf

那么接下来就可以创建一条解析记录，类型为A，名称任意，IP为你的服务器IP

你会发现右边有一个云朵标志，先不要打开

点击保存

### 设置你的节点开启TLS

回到xui面板，打开tls。

随后写上你的域名，选择证书设置的第二项，粘贴证书内容

要注意对应cert和key两个部分，设置好后保存

随后返回cf，在刚刚的解析记录打开云朵标志，保存

过一段时间，ping一下节点，发现有了数值（非-1）

即说明你的搭建成功了
